Introduction to HTML

HTML stands for Hypertext Markup Language, and it is the standard language used for creating web pages.

Basic HTML Tags

  • <html>: The root element of an HTML document.
  • <head>: Contains metadata about the document.
  • <body>: Contains the visible content of the document.

What is an HTML Tag?

An HTML tag is a set of characters used to define elements in an HTML document. Tags are enclosed in angle brackets, e.g., <p> for a paragraph.

Did You Know?

HTML was created by Tim Berners-Lee in 1991.
